プライバシーポリシー

このChrome拡張機能（以下、「本拡張機能」）は、ユーザのプライバシーを尊重し、楽天PointClubからポイントの履歴をスクレイピングして取得し、そのデータをCSV形式で保存する目的で提供されています。以下は、本拡張機能に関するプライバシーポリシーです。

1. データの取得と使用

1.1 データ取得: 本拡張機能は、楽天PointClubのポイント履歴確認ページ(https://point.rakuten.co.jp/history)からのみユーザが明示的に指定したアクション（例: ポイントの履歴を表示するクリック）に基づき、スクレイピングを使用してポイントの履歴データを取得します。

1.2 データ使用: 取得したデータは、ユーザが閲覧するためにローカルでのみ使用され、本拡張機能が提供するCSVエクスポート機能を通じてのみ保存されます。データはユーザが指定した保存先にのみ保存され、本拡張機能はそのデータにアクセスすることはありません。

2. データの保存

2.1 永続的な保存: 本拡張機能は、スクレイピングで取得したデータをローカルストレージに永続的に保存することはありません。データの保存は、ユーザがエクスポート機能を使用して明示的に指定した場所にのみ行われます。

3. 通信と外部サービス

3.1 外部通信: 本拡張機能は一切外部と通信しません。楽天PointClubサイト以外の外部サービスやサーバーとのデータ交換は行いません。

3.2 ユーザデータの送信: 本拡張機能は、ユーザのデータを外部に送信することはありません。ユーザのプライバシーは最大限に尊重されます。

4. セキュリティ

4.1 データセキュリティ: 本拡張機能は、ユーザのデータの機密性を確保するために最善の努力を払います。スクレイピングで取得したデータはユーザのデバイス内でのみ処理され、外部への漏洩は防がれます。

5. プライバシーポリシーの変更

5.1 本拡張機能のプライバシーポリシーは変更されることがあります。変更が行われた場合、ユーザは本拡張機能の最新のプライバシーポリシーを確認する責任があります。
