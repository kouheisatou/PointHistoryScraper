楽天ポイント履歴一括保存

楽天PointClubのポイント履歴をローカルに蓄積し、期間指定でCSVエクスポートできるChrome拡張機能

Chrome Web Store GitHub

主な機能

  • ダッシュボード

    獲得・チャージ・利用の合計を全期間・今年で一覧表示。月平均やポイント利息累計も確認できます。

  • 月別推移グラフ

    直近12ヶ月の獲得・チャージ・利用を棒グラフと累計獲得の折れ線で表示。1ヶ月単位でスライドして過去も確認できます。

  • サービス別獲得ポイント

    獲得ポイントをサービス別に円グラフで可視化。スライスをクリックすると履歴の詳細へジャンプします。

  • 期間・サービス絞り込み

    年・月のショートカットボタンやサービス名チップで履歴をワンタップ絞り込み。絞り込んだ状態でCSVエクスポートも可能です。

  • CSVインポート・エクスポート

    過去のCSVファイルを複数選択してインポート。重複は自動排除され、期間指定でCSV出力できます。

  • グラフ画像保存

    月別推移・サービス別グラフをタイトル付きPNG画像としてダウンロード保存できます。

  • 外部送信なし

    取得データはすべて端末内のローカルストレージで処理し、外部サーバーへ送信しません。

使い方

  1. 拡張機能をインストールし、ツールバーのアイコンをクリックするとサイドパネルが開きます。
  2. 「取得開始」ボタンを押すと楽天ポイント履歴ページを自動で巡回しデータを蓄積します。
  3. ダッシュボードタブでグラフやサマリーを確認し、履歴タブで絞り込み・CSVエクスポートできます。
