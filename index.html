楽天ポイント履歴一括保存

楽天PointClubのポイント履歴をローカルに蓄積し、期間指定でCSVエクスポートできるChrome拡張機能

Chrome Web Store GitHub

主な機能

  • local storage 蓄積

    CSVをページ単位で即時ダウンロードせず、履歴をローカルへ保存します。

  • 期間指定CSVエクスポート

    開始日・終了日を指定して、必要な範囲のみCSV出力できます。

  • 全列一致で重複排除

    date / service / title / action / point / note が一致する行を重複と判定します。

  • 外部送信なし

    取得データは端末内で処理し、外部サーバーへ送信しません。

使い方

  1. 拡張機能をインストールします。
  2. 拡張機能ポップアップで「楽天ポイント履歴 取得開始」を押します。
  3. 取得完了後、開始日と終了日を指定し「期間指定でCSVエクスポート」を押します。
